## 0. なぜ今、再探索の価値が復活しているか

fair-gen005 の重み・挙動は**旧世界（時計なし・手札無制限・手札防御無制限）**で最適化されたもの。
本採用でゲームの根本が変わり、評価地形がリセットされた:

- **山札が寿命になった**（衰弱）: `deckOutPressure` 等の資源系重みに初めて本物の意味が生まれた
- **手札上限 6**: 手札価値（`handCard`）は 6 枚を超えると無価値。溜め込み評価の上限が必要
- **ライフ判定**: 40 ターン到達時はライフ差が勝敗。終盤はライフ差そのものが「勝ち点」になった
- **手札防御 power3 以下**: power4 攻撃は必ず通る。攻撃評価の景色が変わった
  （公開推定は既に `handDefenseMaxPower` を読むが、重みバランスは旧世界のまま）

過去の「重み探索は全滅」の実績は**旧地形の結果であり無効**（プランナー導入時と同じ理屈）。

## 3. トラック W2 — 時計世界の新評価特徴（1 特徴 = 1 実験）

いずれも「重み 0 = 現行とビット一致」から出発し、単体テスト + ガントレットで検証:

1. **衰弱クロック評価**: 自分/相手の「衰弱開始までの残りターン」を評価に入れる
   （山札枚数とドロー速度から計算可能。公開情報のみ）
2. **手札上限評価**: 手札価値を 6 枚で頭打ちにする（超過分のドロー・サーチを減点）
3. **ライフ判定エンドゲーム**: 残りターンが少ない局面でライフ差の重みを増す
   （ターン数は公開情報。「勝ってる時は流し、負けてる時は勝負に出る」挙動の獲得）
4. **power4 攻撃の再評価**: 手札防御不可を攻撃価値に明示反映（公開推定経由で部分的に
   織り込み済みのため、追加の直接ボーナスが有効かを A/B）

# コイン・コレクション仕様

新弾（第2弾以降）のカード入手とパック開封に関する正本仕様。対戦ルールは `game-spec.md` を参照。

## コイン

| 項目 | 値 |
| --- | --- |
| 初期所持コイン | 10 |
| 勝利報酬 | 10 コイン |
| 敗北・引き分け報酬 | 5 コイン |
| パック開封コスト | 5 コイン |

- 報酬は CPU 戦を最後まで打った試合（ライフ 0、リソース切れ、または最大手番のライフ判定で決着）にのみ付与する。
- 途中で「再戦」した試合とチュートリアルは報酬対象外。
- 残高は `localStorage`（キー: `break-duel:wallet`）に保存する。

## パック開封

- 1 パック 5 コイン・5 枚入り。中身は開封（封を剥いた）瞬間に抽選する。開封前の引き直しはできない。
- スロット構成: 1〜3 枚目 = N、4 枚目 = R、5 枚目 = R（60%）または SR（40%）。SR はさらに 25% で SECRET に昇格（全体約 10%）。
- レアリティは暫定的にカード属性から導出する: power 1–2 召喚獣・術式 = N、power 3 召喚獣・遺物 = R、power 4 召喚獣 = SR。
- 高レア（SR / SECRET）入りのパックは、開封の瞬間に確定演出（光柱）が発生する。

## コレクション

- 開封したカードは所持コレクション（キー: `break-duel:collection`、cardId → 所持枚数）に記録する。
- 第 1 弾（`Card.set` 未指定）のカードはスターター扱いで、各カードを 2 枚所持しているものとして表示・使用する。
- 第 2 弾以降（`Card.set >= 2`）のカードは、**所持枚数までしかデッキに入れられない**（デッキ保存時に検証。同名 2 枚の上限は従来通り併用）。

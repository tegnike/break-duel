状態: **実装済み・バランス検証済み（2026-07-06）**。30 種のカード仕様は `game-spec.md`（5.1.1 / 5.2 / 5.3 / 6.5）に移記済み。リーグ戦バランス検証は完了（結果は `docs/balance-history.md` 2026-07-06 エントリ）。同日、ユーザー判断でデュアル属性4種を単属性代替4種に差し替え、再検証済み（同 `balance-history.md` エントリ参照）。さらに同日、ユーザーレビューを受けたカード効果修正13件（水軸弱体化6枚含む）を反映して再検証済み（`echoes` 54.1% / `water` 50.0% で目標維持、デッキ調整不要。同 `balance-history.md` 追加エントリ参照）。残タスクはカードアート生成、パック抽選の第2弾プール接続。

## 設計方針

- 3テーマ混成: トラッシュ・シナジー[T] / チャージ拡張[C] / 遺物シナジー[R]
- **デュアル属性[D] は将来の弾に延期（2026-07-06 ユーザー判断）**: 当初のデュアル属性カード4種（溶岩甲ヴァルカ / 蒸気竜スチマー / 霧幻蝶ルウ / 砂嵐狼ザラン）は実装後に「第2弾では時期尚早」と判断され、単属性の power 3 代替4種（焔角のグレンド / 深響のセレナ / 翠嵐鷹ハヤテ / 古磐熊ゴロン）に差し替え済み。エンジンのデュアル属性基盤（`subAttribute` / `hasAttribute` 等、TS+Python）は使用カードゼロのまま予約仕様として残す
- 水属性を最重点で強化（新エース、専用ドロー術式、防御リターンの新しい顔）
- 他TCG参考の「攻撃強化術式」を新規領域として追加
- レアリティは既存導出を踏襲: power1-2召喚獣・術式 = N、power3召喚獣・遺物 = R、power4召喚獣 = SR
- ID規則: 第2弾通常召喚獣は `D` 系統（例 `AI-FIRE-2D`）、チャージ対応は既存 `C` 系統の power 拡張（`AI-WIND-3C` 等）。全カード `set: 2`

## バランス目標（承認済み・2026-07-06 / 同日緩和改訂）

- **新弾はやや強め**が公式方針: 第2弾の新カードを勝ち筋にしたデッキが、既存プリセット相手のリーグで**勝率 50% 台（既存デッキよりやや強い水準）**に乗ること。52〜56% は目安であり厳守ではない（多少の上振れ・下振れは許容）。**60% を超える構成に落ち着く場合は即不合格ではなく、その状態で作業を止めてユーザー確認を仰ぐ**（勝率表・ワンサイド率・原因カードを添えて報告し、採用可否はユーザーが判断する）。ワンサイド化の悪化は引き続き回避する
- 第2弾を看板にしたプリセットデッキを最低 1 つ追加し（`echoes` / 残響胎動デッキ）、既存 7 デッキとの総当たりで検証する。**第1弾カードとの混合構成でよい**（第2弾比率の下限は設けない）が、第2弾の新カードがデッキの特徴・勝ち筋として機能していること（新弾を引く意味が伝わる構成）は維持する
- 水を最強格に寄せる（グランマーレ＝手札戻り＋ドロー、深流呼び＝実質2枚アド）。水単も 50% 台前半へ上方修正（第2弾の水カードを水単に足す調整も可）
- `貫きの眼光` と `天嵐王ジェイル` は壊れ筆頭候補。シミュレーションで最初に監視する
- 検証は `.agents/skills/ai-break-duel-balance-tuning` の手順（2シード以上のリーグ＋excitement指標＋balance-history追記）に従う
- 2026-07-06 実施結果: `echoes` デッキ再構成＋`残響召喚` 緩和（power 2 → 3）＋水単への `潮汲みモネ` 投入で検証済み。数値は `docs/balance-history.md` 同日エントリを参照
- 2026-07-06 追加検証: ユーザーレビュー反映13件のカード効果修正（水軸6枚の強化/弱体化含む）後も echoes 54.1% / water 50.0% で目標レンジを維持し、デッキ再構成は不要と判断。60% 超は発生せず。数値は `docs/balance-history.md` 同日「ユーザーレビュー反映13件のカード効果修正後の再検証」エントリを参照

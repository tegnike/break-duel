# v0.9.0 (2026-07-11)

対戦相手のキャラクターとプレイスタイルを管理し、対戦前に選べるようになりました。
立ち絵、デッキ、CPU難易度、セリフ、ボイスを組み合わせた独自の対戦相手をローカルに登録できます。

## ハイライト

- **対戦相手のキャラ管理を追加**: カスタムキャラクターを作成・編集・削除できるPC向け管理画面を追加しました。
- **キャラごとに対戦設定**: 使用デッキ、CPU難易度、立ち絵、15種類のセリフとボイスを登録できます。
- **対戦準備画面を刷新**: 自分の使用デッキと対戦キャラクターを確認してからバトルを開始できるようにしました。
- **対戦中の表現をキャラ化**: 相手名、ポートレート、セリフ、ボイス、カットインが選択したキャラクターに合わせて切り替わります。

## 変更点

### 新機能・カード

- 組み込みキャラクター「ニケ」をベースに、任意の名前とキャラクターIDでカスタム対戦相手を登録できるようにしました。
- 通常、被弾、喜び、切札、とどめの各立ち絵と、対戦開始から勝敗までの15種類のセリフ・音声を設定できるようにしました。
- キャラクター素材をブラウザのローカル保存領域に保存し、次回起動時も再利用できるようにしました。
- 保存済みデッキを自分または対戦相手の使用デッキとして選べるようにしました。
- チュートリアルは従来どおり、固定のニケと初心者CPUで遊ぶ構成を維持しました。

### 修正

- 既存キャラクターと同じIDで新規登録した際に、保存データを上書きしないようにしました。
- キャラクター定義の読み込みに失敗した場合も、直前の正常なカタログを維持するようにしました。
- 未対応の将来バージョンのデッキJSONを、現行形式として誤って読み込まないようにしました。

### その他

- 対戦相手プロファイル、キャラクター素材、対戦開始処理の保存・検証テストを追加しました。
- カスタムキャラクター素材とゲーム内テキストのライセンス対象を明確化しました。
- `package.json` / `package-lock.json` のバージョンを `0.9.0` に更新しました。

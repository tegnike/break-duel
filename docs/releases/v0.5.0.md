# v0.5.0 (2026-07-08)

ライバルの山場行動に、全画面カットイン演出を追加したリリースです。
切札の登場と、とどめの一撃がより分かりやすく、対戦の盛り上がりとして伝わるようになりました。

## ハイライト

- **ライバルカットイン追加**: ライバルが切札を出す時と、とどめの攻撃を決める時に、専用立ち絵・帯演出・セリフを表示します。
- **カットイン専用ボイス/画像を追加**: `trump` / `finisher` 用のライバルボイスと立ち絵を追加しました。
- **演出の流れを整理**: 相手 power 4 の登場・アップグレードは、従来の peak リール強調からカットイン演出へ置き換えました。
- **Seed 表示を DEV へ移動**: 通常のヘッダーから開発用の Seed 表示を外し、DEV パネル内で確認できるようにしました。

## 変更点

### 新機能・カード

- ライバルの `trump` カットインを追加し、相手が power 4 カードを場に出す/アップグレードする直前に再生するようにしました。
- ライバルの `finisher` カットインを追加し、相手の攻撃で致死ダメージが確定した時に再生するようにしました。
- カットイン表示用の `DuelCutInOverlay`、演出タイミング、効果音/ボイス再生、演出プレビューを追加しました。
- DEV パネルにカットインプレビュー操作と現在 Seed の表示を追加しました。

### 修正

- カットインと防御パネルの重なり順を調整し、防御選択UIと重なる場面でもカットインが前面に出るようにしました。
- カットイン文言を共通定数から参照するようにし、画面表示とボイスラインの文言がずれないようにしました。
- 通常プレイ時のヘッダーから開発用 Seed 表示を削除し、プレイヤー向けUIのノイズを減らしました。

### その他

- 相手カットイン演出の仕様を `docs/game-spec.md` に追記しました。
- カットイン判定、オーバーレイ表示、カード効果カバレッジのテストを追加・更新しました。
- `package.json` / `package-lock.json` のバージョンを `0.5.0` に更新しました。

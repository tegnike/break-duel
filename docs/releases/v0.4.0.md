# v0.4.0 (2026-07-08)

対戦画面の検証効率を上げる DevPanel と、召喚着地時にカード素材そのものが反応する属性演出を追加したリリースです。
あわせて CPU の終盤挙動、最大ターン時の引き分け処理、カード表示の細部を整えました。

## ハイライト

- **DevPanel 追加**: 開発ビルド限定で、対戦中に手札・場・メモリー・山札・トラッシュ、ライフ、ターン、アクション、勝敗演出を直接調整可能にしました。
- **召喚着地演出を刷新**: 火・水・風・土の属性ごとに、カード上で炎、波紋、風、ひび割れが発生する演出へ作り直しました。
- **属性 SFX 追加**: 召喚獣と遺物の着地に、WebAudio 生成の属性別効果音とカード発光を重ねました。
- **終盤処理を安定化**: CPU の終盤攻撃、最大ターン到達時の引き分け、ライフ下限表示、カード比率を修正しました。

## 変更点

### 新機能・カード

- DEV ビルド限定の盤面編集パネルを追加し、検証したい盤面や勝敗演出を素早く再現できるようにしました。
- 召喚獣の着地演出を、属性ごとにカード素材が反応する canvas 演出へ刷新しました。
- 召喚獣・遺物の着地時に、属性色バースト、カード発光、属性別 SFX を追加しました。

### バランス調整

- 最大ターン到達時はライフ判定ではなく引き分けとして処理するように統一しました。
- CPU が終盤やリソース切れ付近でも適切に攻撃し、最大ターン時の決着処理が安定するよう調整しました。

### 修正

- 対戦中のライフ表示が下限を下回らないよう修正しました。
- カード表示の縦横比と power 4 の切り札イベント表示を調整しました。
- ルール説明の最大ターン数を設定値から表示するようにし、将来の設定変更と文言がずれないようにしました。
- DevPanel のカード削除処理で、無効な負数 index が配列末尾を削除しないようにしました。

### その他

- DevPanel、召喚演出、カード効果、ターン行動、Python ルールの回帰テストを追加・拡充しました。
- アーキテクチャ、ゲーム仕様、バランス履歴、develop PR 作成スキルを更新しました。
- CodeRabbit 指摘を反映し、スキル文書の環境依存パスを削除しました。

# v0.6.0 (2026-07-10)

パック開封の高レア演出を大きく強化し、カードを引く瞬間の手応えを上げたリリースです。
あわせて、バランス検証と自動対戦の基盤を TypeScript に統一し、今後の調整をよりゲーム本体に近い条件で行えるようにしました。

## ハイライト

- **パック開封演出を刷新**: SR/UR の予兆、光条、粒子、結果表示を強化し、高レアカードの出現がより分かりやすくなりました。
- **UR ホロ表現を改善**: カード表示のホロ演出を整理し、パック開封画面とカード表示の見え方を揃えました。
- **シミュレーション基盤を TypeScript に統一**: 自動対戦、リーグ戦、コストバランス検証をブラウザ本体と同じ実装で動かす構成にしました。
- **ルール回帰テストを拡充**: 旧 Python テストを Vitest へ移植し、AI・防御・攻撃・アップグレード・デッキ構築の検証を増やしました。

## 変更点

### 新機能・改善

- パック開封時のレア予兆、結果コールアウト、UR カットイン、粒子エフェクトを強化しました。
- 第2弾コレクション進行表示とパック結果画面の見た目を調整しました。
- `npm run sim`、`npm run balance:cost`、`npm run tune:ai`、`npm run tune:apex` を TypeScript 実装に統一しました。

### 修正

- パック開封ボタン文言と演出位置のずれを修正しました。
- 高レア演出の負荷を抑え、canvas 描画量を調整しました。
- CodeRabbit 指摘に基づき、balance regression workflow の入力処理を安全化しました。

### その他

- Python シミュレータ本体、旧 Python テスト、旧 Python 探索スクリプトを削除しました。
- ルール、アーキテクチャ、バランス履歴、最強 CPU 計画ドキュメントを TypeScript 統一後の内容へ更新しました。
- `package.json` / `package-lock.json` のバージョンを `0.6.0` に更新しました。

## 検証

- `npm run check` 全合格（typecheck / Vitest 291件 / build）
- PR #30 の CI `check` 成功、CodeRabbit 成功
- main ブランチの CI 成功、Deploy 成功

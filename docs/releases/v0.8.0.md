# v0.8.0 (2026-07-10)

第2弾パックをまとめて楽しめる10連開封を追加しました。
購入した10パックを順番に開封し、最後に全50枚とレアカード、新規入手カードをまとめて確認できます。

## ハイライト

- **10連パック開封を追加**: 50コインで10パック・合計50枚をまとめて購入できるようになりました。
- **開封結果をまとめて確認**: 10パック目の開封後に、全カードと `NEW`・SR・URの集計、コレクション進捗を一覧表示します。
- **パックごとの演出を維持**: 購入後も1パックずつカードをめくり、次のパックへ進む開封体験を楽しめます。

## 変更点

### 新機能・カード

- パック開封前に「単発」と「10連」を選べるようにしました。
- 10連では、各パックを単発と同じ排出率で独立抽選します。
- 各パックの5枚を公開したあと、結果を確認してから次のパックへ進めるようにしました。
- 全10パックの開封後に、50枚をパック単位で確認できるスクロール一覧を追加しました。
- 10連結果に新規入手数、SR数、UR数、第2弾コレクション進捗を表示するようにしました。

### 修正

- 10連パック内で次のパックへ進んだ際も、レアカードの予兆演出が正しく維持されるようにしました。

### その他

- 対戦、カード一覧、デッキ製作、パック開封が共有するカードカタログを一元化し、カード定義の不整合を防ぐ構成にしました。
- カードカタログと10連パック抽選・表示に関するテストを追加しました。
- `package.json` / `package-lock.json` のバージョンを `0.8.0` に更新しました。

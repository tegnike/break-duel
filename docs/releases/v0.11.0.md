# v0.11.0 (2026-07-11)

人間とCPUの対戦を、AI強化へ活用できるローカル戦闘ログとして残せるようになりました。

## ハイライト

- 通常の人間対CPU戦を、1対局1ファイルの構造化JSONLとして自動保存
- デッキ、手札、場、ライフ、行動、防御、チャージ、攻城状態を時系列で記録
- 終了時送信、失敗時再試行、重複排除、入力検証によりログの信頼性を強化

## 変更点

### 新機能

- ローカル開発サーバーで行った通常対戦を `tmp/human-battle-logs/` に保存
- AIが人間の選択とCPUの候補手を比較できる完全情報スナップショットを追加
- 対戦終了、途中離脱、ページ終了を区別して記録

### その他

- Backlog.mdによる開発タスク管理を導入
- 新機能を専用worktreeで進める開発手順を文書化

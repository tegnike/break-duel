# v0.12.0 (2026-07-13)

対人戦のCPUが攻撃機会を正しく評価し、アップグレードや手札超過時にも意図した行動を取りやすくなりました。

## ハイライト

- CPUが手札超過時に行動できる場面で即ターン終了してしまう挙動を修正
- アップグレード登場したPower 3/4が、ルールどおり未消耗で行動できる状態へ復旧
- CPUの先読みが人間側の防御を公開情報から評価し、攻撃や連続攻撃を正しく候補にできるよう改善

## 変更点

### CPU・対戦挙動

- 手札上限を超えたCPUが、未使用アクションで手札を減らせる場合は行動を続けるよう修正
- CPUの計画探索中だけ、人間側の防御を公開情報に基づく推定で解決し、実際の対戦画面では従来どおり人間が防御を選択する構造を維持
- 対人戦でCPUが攻撃を過小評価していた問題を解消し、囮攻撃から本命を通す複数手の計画も評価可能に変更

### 修正

- 通常登場時のPower 3/4消耗ルールが、アップグレード登場にも誤適用されていた回帰を修正
- CPU経路と人間操作経路のアップグレード処理を共通化し、アップグレード後は未消耗になる仕様を統一
- 対人戦ログ診断で、旧形式ログのアクション解決状態を復元できるよう改善

### その他

- 対人戦ログからCPUの候補手、スコア、ビーム探索結果を再現する診断ツールを追加
- CPU評価の将来実験に使う特徴量を、現行挙動へ影響しない重み0で追加
- CPU再検証結果、バランス履歴、関連仕様の監査結果を文書化

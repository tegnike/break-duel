# v0.1.0 (2026-07-05)

Break Duel の最初のリリースです。ブラウザでCPUと対戦できるファンタジーカードゲームとして、
カードプール60種・チュートリアル・デッキ制作・CPU 2種を含むゲーム本体一式を収録しています。

## ハイライト

- **60種のカードプール**: 火・水・風・土の4属性に召喚獣（power 1〜4）、術式、遺物、チャージ系統を収録
- **全22ステップのチュートリアル**: 通常ルールそのまま（先攻1ターン目は1アクション・ライフ8）で、召喚から切札のとどめまでを固定進行で案内
- **モンスター攻撃と手札防御**: 攻撃対象に相手召喚獣を選べ、手札防御はプレイヤーへの攻撃も召喚獣への攻撃も1ターン1回まで割り込み可能
- **2種のCPU**: 手加減あり構造の「初心者」と、合法手評価で戦う「挑戦者」

## 収録内容

### ゲームルール
- ライフ8・毎ターン3アクション制（先攻1ターン目のみ1アクション）
- 召喚コスト = power、場の中型を元にした大型アップグレード
- チャージ（手札1枚→アクション+1、当該ターン攻撃不可）
- 場防御・手札防御（1ターン1回）・防御しない選択と被弾時ブレイクドロー
- power 4 の攻撃後退場（オーバーヒート）、リソース枯渇による強制敗北

### カード・デッキ
- 4属性×power 1〜4 の召喚獣（B系統・チャージC系統含む）、属性術式、遺物カード
- 固定デッキ7種（紅蓮突破・大地守護・単色4種・覇王結束）と25枚デッキ制作画面
- カード一覧ページと生成カードアート

### UI・演出
- 対戦準備モーダル、アクションドック、対象選択UI、トラッシュ演出、BGM・ボイス

### 開発基盤
- Python シミュレータ（`ai_break_duel/`）と TypeScript 実装のルール二重検証
- プリセットデッキ回帰・偏りコストストレスデッキによるバランス検証記録（`docs/balance-history.md`）
- GitHub Actions による CI（typecheck / vitest / build / Python unittest）と Deploy

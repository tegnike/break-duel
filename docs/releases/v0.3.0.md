# v0.3.0 (2026-07-07)

第2弾「残響の胎動」を中心に、カードプール、パック開封、コレクション、
防御ルール、カード演出を大きく拡張したリリースです。新デッキ `echoes`
と水・トラッシュ・チャージ・遺物軸の新カードで、対戦とデッキ作成の選択肢が増えました。

## ハイライト

- **第2弾「残響の胎動」追加**: 新カード30種、`echoes` プリセットデッキ、ターン限定攻撃強化、蘇生、遺物破壊/回収などを追加
- **パック開封とコレクション実装**: 対戦報酬コイン、5枚入りパック、カード所持数、デッキ保存時の所持枚数チェックを追加
- **場防御の遊びやすさを改善**: モンスター攻撃への場防御、場防御失敗時の差分ダメージ、場防御時効果の発動条件を整理
- **カード演出を強化**: レアリティ、3Dチルト、グレア、SR/UR演出、パック開封音とレア演出音を追加

## 変更点

### 新機能・カード

- 第2弾カード30種を追加し、TypeScript / Python 両方のゲームエンジンとテストへ同期
- 新プリセットデッキ `echoes`（残響胎動）を追加
- コイン報酬、ウォレット、コレクション保存、パック開封画面を追加
- 第2弾カード向けのカードアート、パック画像、効果音、パック用BGMを追加
- デッキ工房にセット別表示と所持カードに応じたデッキ保存チェックを追加

### バランス調整

- モンスター攻撃に対して、対象以外の未消耗召喚獣で場防御できるように変更
- 場防御失敗時のダメージを、攻撃値そのままではなく `攻撃値 - 防御値` の差分に変更
- `CMD-GRAVE-CALL`（残響召喚）の蘇生対象を power 2 以下へ調整
- `echoes` デッキの遺物構成を調整し、差分ダメージ環境での過剰な上振れを抑制
- 第2弾カードのユーザーレビュー反映後も、`echoes` / `water` の勝率が目標範囲に収まることを再検証

### 修正

- 場防御時効果、手札防御、攻撃バフ、CPU 防御選択の処理を新ルールに合わせて修正
- デッキビルダーのカード可用性表示とカード一覧下部の表示崩れを修正
- パック開封のキーボード操作時に不要な効果音が鳴らないよう調整
- `Array.at` に依存していたテストを修正し、CI 環境で安定するように調整

### その他

- CodeRabbit の review 対象に `main` を追加し、`develop -> main` の昇格 PR も確認対象に変更
- デスクトップブラウザのみを対象とする UI 開発方針を `AGENTS.md` / `CLAUDE.md` に明記
- 独自ライセンスの整理を README / LICENSE / LICENSE-ASSETS に反映

# v0.10.0 (2026-07-11)

スマホブラウザでも、端末を横向きにして Break Duel を遊べるようになりました。
PC向けの画面設計を保ちながら、スマホでは横長キャンバスを端末に合わせて表示します。

## ハイライト

- **スマホ横向きプレイに対応**: スマホブラウザではPC用キャンバスを端末幅へ縮小し、横向きで対戦できるようにしました。
- **縦向き時の案内を追加**: スマホを縦向きで開いた場合、横向きへの回転を促す専用画面を表示します。
- **横向き切り替えを補助**: 対応ブラウザでは、全画面表示と画面方向のロックをボタンから試せるようにしました。

## 変更点

### 新機能・カード

- iPhone、iPod、Androidなどのスマホブラウザを判定し、スマホ専用の横向き表示制御を追加しました。
- スマホ横向き時にゲーム画面、モーダル、デッキ編集、キャラクター管理を端末の高さに合わせて縮小表示するスタイルを追加しました。
- 画面方向の自動変更に対応しないブラウザでも、手動回転の案内を表示してプレイを継続できるようにしました。

### 修正

- スマホブラウザの全画面化や画面方向ロックが失敗しても、操作不能にならず案内へフォールバックするようにしました。
- タブレットやデスクトップ環境をスマホ用表示へ誤判定しにくい端末判定にしました。

### その他

- スマホ判定、全画面化、画面方向ロック、フォールバック表示のユニットテストを追加しました。
- UI対応方針を「PC優先・スマホ横向き限定」に更新しました。
- `package.json` / `package-lock.json` のバージョンを `0.10.0` に更新しました。
